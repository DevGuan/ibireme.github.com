YYAnimatedImage Protocol Reference

Conforms to NSObject
Declared in YYAnimatedImageView.h

Overview

The YYAnimatedImage protocol declares the required methods for animated image display with YYAnimatedImageView.

Subclass a UIImage and implement the protocol’s method, so that instances of that class can be set into a YYAnimatedImageView to display animated image.

see YYGIFImage for example.
