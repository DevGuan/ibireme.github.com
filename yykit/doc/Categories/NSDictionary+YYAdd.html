Class Methods

dictionaryWithXML:

Try to parse an XML and wrap it into a dictionary. If you just want to get some value from a small xml, try this.

+ (NSDictionary *)dictionaryWithXML:(id)xmlDataOrPath

Parameters

xmlDataOrPath

XML in NSData or NSString format.

Return Value

Return a new dictionary, or nil if an error occurs.

Discussion

example XML: “link” example Return: @{@“name”:@“config”, @“a”:{@“text”:@“link”,@“href”:@“test.com”}}

allKeysSorted

Returns a new array containing the dictionary’s keys sorted. The keys should be NSString, and they will be sorted ascending.

- (NSArray *)allKeysSorted

Return Value

A new array containing the dictionary’s keys, or an empty array if the dictionary has no entries.

allValuesSortedByKeys

Returns a new array containing the dictionary’s values sorted by keys.

- (NSArray *)allValuesSortedByKeys

Return Value

A new array containing the dictionary’s values sorted by keys, or an empty array if the dictionary has no entries.

Discussion

The order of the values in the array is defined by keys. The keys should be NSString, and they will be sorted ascending.
