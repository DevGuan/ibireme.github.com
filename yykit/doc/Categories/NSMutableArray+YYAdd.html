insertObjects:atIndex:

Adds the objects contained in another given array at the index of the receiving array’s content.

- (void)insertObjects:(NSArray *)objects atIndex:(NSUInteger)index

Parameters

objects

An array of objects to add to the receiving array’s content. Nothing will happen if the objects is empty or nil.

index

The index in the array at which to insert objects. This value must not be greater than the count of elements in the array. Raises an NSRangeException if index is greater than the number of elements in the array.
