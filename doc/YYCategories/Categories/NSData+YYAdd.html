aes256DecryptWithkey:iv:

Returns an decrypted NSData using AES.

- (NSData *)aes256DecryptWithkey:(NSData *)key iv:(NSData *)iv

Parameters

key

A key length of 16, 24 or 32 (128, 192 or 256bits).

iv

An initialization vector length of 16(128bits). Pass nil when you don’t want to use iv.

Return Value

An NSData decrypted, or nil if an error occurs.
