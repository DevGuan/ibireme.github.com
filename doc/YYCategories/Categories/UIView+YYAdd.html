convertPoint:fromViewOrWindow:

Converts a point from the coordinate system of a given view or window to that of the receiver.

- (CGPoint)convertPoint:(CGPoint)point fromViewOrWindow:(UIView *)view

Parameters

point

A point specified in the local coordinate system (bounds) of view.

view

The view or window with point in its coordinate system. If view is nil, this method instead converts from window base coordinates.

Return Value

The point converted to the local coordinate system (bounds) of the receiver.
