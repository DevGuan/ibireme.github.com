postNotificationOnMainThread:

Posts a given notification to the receiver on main thread. If current thread is main thread, the notification is posted synchronized; otherwise, is posted asynchronized.

- (void)postNotificationOnMainThread:(NSNotification *)notification

Parameters

notification

The notification to post. An exception is raised if notification is nil.

postNotificationOnMainThread:waitUntilDone:

Posts a given notification to the receiver on main thread.

- (void)postNotificationOnMainThread:(NSNotification *)notification waitUntilDone:(BOOL)wait

Parameters

notification

The notification to post. An exception is raised if notification is nil.

wait

A Boolean that specifies whether the current thread blocks until after the specified notification is posted on the receiver on the main thread. Specify YES to block this thread; otherwise, specify NO to have this method return immediately.
