UIColor(YYAdd) Category Reference

Declared in UIColor+YYAdd.h

Overview

Provide some method for UIColor to convert color between RGB,HSB,HSL,CMYK and Hex.

Color space Meaning
RGB * Red, Green, Blue
HSB(HSV) * Hue, Saturation, Brightness (Value)
HSL Hue, Saturation, Lightness
CMYK Cyan, Magenta, Yellow, Black

Apple use RGB & HSB default.

All the value in this category is float number in the range 0.0 to 1.0. Values below 0.0 are interpreted as 0.0, and values above 1.0 are interpreted as 1.0.

If you want convert color between more color space (CIEXYZ,Lab,YUV…), see https://github.com/ibireme/yy_color_convertor

colorWithHexString:

Creates and returns a color object from hex string.

+ (UIColor *)colorWithHexString:(NSString *)hexStr

Parameters

hexStr

The hex string value for the new color.

Return Value

An UIColor object from string, or nil if an error occurs.

Discussion

@discussion: Valid format: #RGB #RGBA #RRGGBB #RRGGBBAA 0xRGB … The # or “0x” sign is not required. The alpha will be set to 1.0 if there is no alpha component. It will return nil when an error occurs in parsing.

Example: @“0xF0F”, @“66ccff”, @“#66CCFF88”
